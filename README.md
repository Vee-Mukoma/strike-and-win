# Strike & Win: Rock Paper Scissors

> Think fast. Strike hard. Will you outsmart the game or go down in flames? Every move counts.

## About

Strike & Win is a browser-based Rock Paper Scissors game built with pure HTML, CSS and JavaScript. Challenge the computer, track your score in real time, and see if your instincts are unbeatable.

## Features

- Neon-themed UI with purple and pink accents
- Animated intro screen with a Start Game button
- Real-time score tracking — Human vs Computer
- Fully responsive — works on desktop and mobile
- Clean, semantic HTML structure

## Tech Stack

- HTML5
- CSS3 — custom properties, flexbox, grid, neon glow effects
- JavaScript — DOM manipulation, event listeners, game logic

## How to Play

1. Click **Start Game** to begin
2. Choose your move — Rock, Paper or Scissors
3. The computer picks randomly
4. The result and scores update instantly
5. Keep playing and see how high your score goes!

## Rules

| Player | Computer | Result |

| Rock | Scissors | Player Wins |
| Scissors | Paper | Player Wins |
| Paper | Rock | Player Wins |
| Same | Same | Tie |
| Any other combo | | Computer Wins |
